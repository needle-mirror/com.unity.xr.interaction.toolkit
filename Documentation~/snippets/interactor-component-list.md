| **Interactor**             | **Description**         |
| :-------------------- | :----------------------- |
| [Near-Far Interactor](xref:xri-near-far-interactor)                          | Interacts with both close and distant interactables when the user either touches or points at them. |
| [XR Direct Interactor](xref:xri-xr-direct-interactor)                        | Interacts with nearby interactables when the user touches them. |
| [XR Poke Interactor](xref:xri-xr-poke-interactor)                            | Interacts with nearby interactables when the user performs a specific poking motion.|
| [XR Ray Interactor](xref:xri-xr-ray-interactor)                              | Interacts with interactables at a distance when the user points at them. |
| [XR Gaze Interactor](xref:xri-xr-gaze-interactor)                            | Interacts with gaze-enabled interactables via eye tracking. (Can fallback to use head tracking for devices that don't support eye tracking.)|
| [XR Socket Interactor](xref:xri-xr-socket-interactor)                        | A specialized interactor that only interacts with interactables close to it. |
| [AR Gesture Interactor](xref:xri-ar-gesture-interactor)                      | (Deprecated) Interacts with `ARGestureInteractable` objects through mobile device touchscreen gesture input. To replace this AR-specific interactor, you can use touchscreen input with the other interactors. The toolkit's Starter Assets provide a set of **Touchscreen Gestures** in the **XRI Default Input Actions** asset. Refer to [AR Interaction Overview](xref:xri-ar-interaction) for more information. |
